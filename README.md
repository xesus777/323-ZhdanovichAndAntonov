## Вариант задания

### Функция 1

w = |cos x - cos y|^(1+2sin²y) · (1 + z + z²/2 + z³/3 + z⁴/4)



### Функция 2

d = { (f(x)-y)³ + arctg(f(x)), x > y;
(y-f(x))³ + arctg(f(x)), y > x;
(y+f(x))³ + 0.5, y = x }


где f(x) может быть:
- sh(x) - гиперболический синус
- x² - квадрат числа
- e^x - экспонента

### Функция 3

y = ax³ + cos²(x³ - b)


Параметры:
- a = 1.35
- b = -6.25
