## Идея проекта
Программный продукт с графическим интерфейсом на базе WPF (.NET 8), предназначенный для автоматизации вычислений трех различных математических выражений. В приложении реализована система навигации между страницами, каждая из которых отвечает за расчет конкретной функции с возможностью визуализации результатов.

### Ключевые функции
- Расчет математических выражений по заданным формулам
- Построение динамических графиков с помощью библиотеки LiveCharts
- Вывод промежуточных и итоговых значений в табличной форме
- Переключение между режимами работы через основное меню
- Гибкая настройка исходных параметров расчета
- Переключение вариантов подставляемой функции f(x) с помощью переключателей

Страница 3 — Табуляция и график y
text
y = ax³ + cos²(x³ - b)
Алгоритм работы:

Задать начальное x₀, конечное xk и шаг dx

Указать коэффициенты a и b

После нажатия «Вычислить»:

Формируется таблица пар (x, y)

Автоматически строится график зависимости

Стандартные настройки:

Параметр	Базовое значение
Начало интервала (x₀)	-1.5
Конец интервала (xk)	10.3
Шаг (dx)	0.25
Коэффициент a	1.35
Коэффициент b	-6.25
Вероятные трудности
При расширении интервала или уменьшении шага количество точек растет, что может замедлить отрисовку графика — рекомендуется подбирать оптимальные параметры

Ввод дробных чисел возможен как с точкой, так и с запятой — приложение корректно обрабатывает оба варианта

Незаполненные поля или некорректные символы приводят к появлению предупреждения с пояснением
